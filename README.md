# 🚲 AI-Based Bike Pooling System

## 📌 Project Overview
This project is an AI-powered bike pooling system designed to help students share rides efficiently.  
It reduces fuel consumption, traffic congestion, and promotes eco-friendly travel.

---

## 🎯 Features
- 🔐 User Registration & Login
- 🚲 Add Ride Details
- 🤖 AI-Based Ride Matching (K-Means Clustering)
- 📍 Location-Based Grouping
- ⏱ Time Compatibility Matching
- 🌐 Web-based Interface
